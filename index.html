Upside down text is a form of text where the characters are flipped upside down, making it appear as if the text is turned 180 degrees. This is often used for fun, to create visual interest, or to add a playful or quirky element to written content. The process involves replacing each character with a corresponding upside-down version, which can be done manually or through online tools. For example, the word "hello" in upside down text might appear as "ɐǝʃʃo." Some characters have direct upside-down equivalents, while others are approximated using similar-looking symbols.

Reverse text refers to writing text backward, so that the order of the characters is flipped from the original. For example, if you take the word "hello" and reverse it, it becomes "olleh." This is different from upside-down text, where the characters are flipped vertically, whereas reverse text only alters the sequence of the characters. Reverse text is often used for playful purposes, puzzles, or to create a mirror effect in writing.

Mirror text flips the text horizontally, making it appear as if viewed in a mirror. The characters and their order are reversed, creating a reflection effect.
